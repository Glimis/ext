Ext.data

Ext.data主要用于描述or获取数据,是Ext的数据来源(依赖),了解页面交互的对象封装,大致有一下几个部分

1.链接

数据的获取-->Ext.data.Connection

即便是链接也可称为一个对象,我们可以在对象中预定义url,参数等其他信息,在需要的时候进行调用,当然这些参数在需要的时候进行变更也是可以的

Ext.Ajax-->Ext.data.Connection的单例对象

我们可以认为ext基于操作习惯,也可以认为是基于性能优化,Ext.ajax是Ext.data.Connection的一个实例(单例),因此,对Ext.ajax的调用直接将其理解为content对象就对了
